alertas module

class alertas.MisAlertas

Bases: object

Clase que contiene métodos para mostrar cuadros de diálogo de confirmación para acciones sensibles como borrar o actualizar registros.

confirmar_actualizar()

Muestra una alerta de confirmación antes de actualizar el stock.

Devuelve:

True si el usuario confirma, False en caso contrario.

Tipo del valor devuelto:

bool

confirmar_borrar()

Muestra una alerta de confirmación antes de borrar un registro.

Devuelve:

True si el usuario confirma, False en caso contrario.

Tipo del valor devuelto:

bool
